📋 CRM College — Deployment Guide

Is system ko college mein kaise lagayein — complete guide

🚀 Phase 1: Setup (Day 1)
Yeh ek baar ka kaam hai — sirf 30 minutes lagenge
1

📁 File Rakhein Computer Mein

Is HTML file ko college ke computer mein save karo. Koi installation nahi chahiye — sirf file ko CRM-Attendance.html naam se Desktop pe rakhein.

2

🌐 GitHub Pages pe Free Hosting (Recommended)

1. github.com pe free account banao
2. New repository banao: crm-attendance
3. HTML file upload karo
4. Settings → Pages → Enable karo
5. Free URL milegi: yourname.github.io/crm-attendance
✅ Koi bhi student apne phone se access kar sakta hai!

3

🔐 Admin Password Change Karo

File mein adminPwd:'crm@admin2026' ko apna strong password se replace karo. Yeh change karna zaroori hai security ke liye!

👥 Phase 2: Students Add Karo (Day 1–2)
100 students ek baar mein add karo
4

📥 Bulk Upload se 100 Students Add Karo

Admin Dashboard → Bulk Upload tab → Yeh format mein data paste karo:
CRM001,Rahul Sharma,9876543210,BCA Sem 1,BCA
CRM002,Priya Singh,9123456780,BCA Sem 2,BCA
...100 students tak
Upload button dabao — Done!

5

📱 Students Ko ID Card Dein

Har student ko unka unique ID batao (CRM001, CRM002, etc.). Yeh ID students apne phone mein save kar lein. Yahi ID unka login credential hai.

📅 Phase 3: Roz Ka Kaam (Daily)
Ek baar setup hone ke baad attendance automatic hoti hai
6

🤖 Auto Attendance — Student ka kaam

Student apna ID dalke Login karta hai → Automatically PRESENT mark ho jaata hai! Admin ko kuch nahi karna.
Student apna dashboard mein dekh sakta hai: Present/Absent count, Attendance %, Exam eligibility.

7

✏️ Manual Override — Admin ka kaam (agar zaroorat ho)

Agar koi student aaya tha par login nahi kar paya, Admin → Attendance tab → manually Present mark kar do. Records mein "Manual" tag lagega.

🎓 Phase 4: Exam se Pehle (End of Year)
75% criteria automatically calculate hoti hai
8

📈 Year Report Check Karo

Admin → Year Report tab → Saare students ki saal bhar ki attendance ek saath dikhti hai. Eligible/Not Eligible automatically calculate hota hai 75% se.

9

🎓 Exam Eligibility Tab

Admin → Exam Eligibility tab → Export button dabao → CSV file download ho jaayegi → Principal ko bhej do. Jo students RED mein hain woh exam nahi de sakte.

10

📢 Low Attendance Warning

Admin → Auto Messages tab → "Send Warning" dabao → Jinki attendance 75% se kam hai unke liye warning message log ho jaata hai. Principal ko monthly report bhejte rahein.

⚠️ Important Notes
• Data LocalStorage mein save hota hai — same browser, same device par rehta hai
• Agar computer format ho ya browser clear karo toh data delete ho jaayega — monthly CSV export karte rahein!
• For permanent storage: Database (Firebase/MySQL) use karna chahiye — future upgrade
• Admin password strong rakhein aur kisi ko na batayein
• Students ko apna ID card ya slip dein join ke time par

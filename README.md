📌 Project Kya Hai?
Yeh ek web-based Smart Attendance Management System hai jo colleges, schools aur coaching institutes ke liye banaya gaya hai. Iska main goal hai ki students aur teachers ka time bachaye aur attendance process ko fully automatic banaye.
Koi server nahi chahiye. Koi app install nahi karni. Bas link kholo aur kaam shuru!

Features
👨‍🎓 Student Side

Auto Attendance — Student ID se login karo, attendance automatically mark
Live Dashboard — Present days, Absent days, Attendance % ek jagah
75% Tracker — Kitne aur din present rahna hai exam ke liye — live calculate
Monthly Chart — Har mahine ki attendance bar chart mein
ID Recovery — Mobile number se bhula hua ID wapas lo
Help Center — Admin ko seedha message bhejo

🛡️ Admin Side
Attendance Tab — Kisi bhi date ki attendance manually mark karo
Students Tab — Students add, search, filter, delete karo
Bulk Upload — CSV drag & drop se 100+ students ek saath add karo
Records Tab — Poori history dekho with date/time filters
Year Report — Har student ki yearly attendance cards mein
Exam Eligibility — Kaun exam de sakta hai, kaun nahi — instantly
Auto Warning — 75% se neeche students ko automatic warning message
Export CSV — Saara data Excel file mein download karo
Custom Holidays — Apne college ki chutti dates add karo

🏗️ Technology Used
Language    : HTML5, CSS3, JavaScript 
Storage     : Browser LocalStorage (no database, no server)
Hosting     : GitHub Pages (Free)
Fonts       : Google Fonts — Poppins
